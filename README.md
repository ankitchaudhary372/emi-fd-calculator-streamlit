# 💰 EMI & FD Return Calculator

An interactive **financial calculator web application** built using **Python and Streamlit**.

The application helps users:

* Calculate **EMI for Home or Car loans**
* Estimate **Fixed Deposit (FD) returns**

This project demonstrates how Python can be used to build **simple financial tools and interactive web applications**.

# 📊 Features

### 🏠 Home / Car Loan EMI Calculator

Users can calculate their **monthly EMI payments** based on:

* Loan amount
* Interest rate
* Loan tenure

The calculator instantly shows the **monthly EMI amount**.

---

### 🧮 Fixed Deposit (FD) Return Calculator

Users can estimate **FD maturity value** using:

* Investment amount
* Interest rate
* Time period

The calculator computes the **final maturity amount and interest earned**.

# ⚙️ Installation

Clone the repository:

```
git clone https://github.com/yourusername/emi-fd-calculator-streamlit.git
```

Move into the project folder:

```
cd emi-fd-calculator-streamlit
```

Install dependencies:

```
pip install -r requirements.txt
```

---

# ▶️ Run the Application

Run the Streamlit app:

```
streamlit run app.py
```

Open in browser:

```
http://localhost:8501
```

---

# ⚡ Quick Start (Single Command)

```
python -m venv venv && venv\Scripts\activate && pip install -r requirements.txt && streamlit run app.py
